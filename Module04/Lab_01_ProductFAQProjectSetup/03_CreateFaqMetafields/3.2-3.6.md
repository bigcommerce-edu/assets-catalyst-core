Request name:
Create FAQ Metafield

Authorization:
No Auth

Request method:
POST

URL:
```
https://api.bigcommerce.com/stores/{{store_hash}}/v3/catalog/products/:product_id/metafields
```

Headers:

| Header       | Value            |
|--------------|------------------|
| Accept       | application/json |
| Content-Type | application/json |
| X-Auth-Token | {{v3_token}}     |

Body:
```json
{
    "key": "ship-time",
    "value": "{ \"question\": \"How quickly does this product ship?\", \"answer\": \"The product ships within 2 days.\" }",
    "namespace": "FAQ",
    "permission_set": "read_and_sf_access"
}
```
