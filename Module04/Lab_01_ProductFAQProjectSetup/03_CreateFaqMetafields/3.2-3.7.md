Request name:
Create FAQ Metafield

Authorization:
No Auth

Request method:
POST

URL:
```
https://api.bigcommerce.com/stores/{{store_hash}}/v3/catalog/products/:product_id/metafields
```

Headers:

| Header       | Value            |
|--------------|------------------|
| Accept       | application/json |
| Content-Type | application/json |
| X-Auth-Token | {{v3_token}}     |

Body:
```json
{
    "key": "ship-time",
    "value": "{ \"question\": \"How quickly does this product ship?\", \"answer\": \"The product ships within 2 days.\" }",
    "namespace": "FAQ",
    "permission_set": "read_and_sf_access"
}
```

Post-response Script:
```javascript
pm.test("Response is not an error", () => {
    pm.response.to.not.be.error;
    pm.response.to.not.have.jsonBody("errors");
});
pm.test("Response is JSON with data", () => {
    pm.response.to.have.jsonBody("data");
});
pm.test("Response includes a metafield ID", () => {
    pm.expect(pm.response.json().data?.id).to.be.a("number");
});
```
