# Install Catalyst

## Run the Installer

### Run Installer

```shell
corepack enable pnpm && pnpm create @bigcommerce/catalyst@0.21.0 --gh-ref @bigcommerce/catalyst-core@makeswift-1.0.0-rc.11
```

## Run the Dev Server

```shell
cd /path/to/catalyst/project
pnpm run dev
```
