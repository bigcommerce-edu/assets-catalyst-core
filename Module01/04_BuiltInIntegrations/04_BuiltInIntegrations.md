# Built-in Integrations

## Installing with an Integration

**pnpm installation:**

```shell
corepack enable pnpm && pnpm create @bigcommerce/catalyst@latest --gh-ref upstream/integrations/<integration-name>
```

## Makeswift

**Installation:**

```shell
pnpm create @bigcommerce/catalyst@latest --gh-ref @bigcommerce/catalyst-core@makeswift-canary
```
